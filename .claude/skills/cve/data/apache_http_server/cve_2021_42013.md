    - 使用PoC脚本检测目标Apache服务器是否易受攻击
    - 检查Apache版本是否为2.4.49或2.4.50
    - 验证CGI模块是否启用

漏洞利用方法
    - 该漏洞是CVE-2021-41773修复不完整导致的路径遍历攻击
    - 攻击者可以利用特制的URL绕过Alias-like指令配置的目录限制
    - 通过路径遍历访问配置目录之外的文件，如果这些文件未受默认配置"require all denied"保护，攻击可能成功
    - 如果CGI脚本在别名路径上启用，可能导致远程代码执行
