验证漏洞存在方法
    - 检查网站是否使用Country State City Dropdown CF7插件
    - 检查插件版本是否≤2.7.2
    - 尝试通过'cnt'和'sid'参数发送SQL注入payload进行测试

漏洞利用方法
    - Country State City Dropdown CF7 WordPress插件在2.7.2及以下版本中存在SQL注入漏洞
    - 漏洞位于'cnt'和'sid'参数，由于对用户提供参数转义不足和现有SQL查询准备不充分
    - 未经身份验证的攻击者可以向现有查询添加额外的SQL查询，从而从数据库中提取敏感信息
    - 通过向插件端点发送包含恶意SQL payload的'cnt'和'sid'参数来执行SQL注入攻击
    - CVSS评分：9.8（CRITICAL），攻击向量：网络，攻击复杂度：低，权限要求：无，用户交互：无，影响范围：机密性-高，完整性-高，可用性-高
