    - 确认 Fluent Bit 的 HTTP 服务器功能已启用
    - 尝试发送特制的 trace 请求到 Fluent Bit 的 HTTP 接口

漏洞利用方法
    - 此漏洞位于 Fluent Bit 嵌入式 HTTP 服务器的 trace 请求解析过程中
    - 攻击者可以通过发送恶意的 trace 请求触发内存损坏
    - 可能导致拒绝服务、信息泄露或远程代码执行
