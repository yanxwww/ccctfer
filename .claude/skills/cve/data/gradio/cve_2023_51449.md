验证漏洞存在方法
    - 检查目标是否运行Gradio应用（可通过响应头或页面内容判断）
    - 确认Gradio版本是否在受影响范围内（4.0-4.10或<3.33）
    - 检查目标是否未启用身份验证

漏洞利用方法
    - 这是一个本地文件包含（LFI）漏洞，影响未启用身份验证的Gradio应用
    - 攻击者可以通过上传文件获取下载路径，然后利用路径遍历漏洞读取服务器上的任意文件
    - 漏洞允许未经身份验证的攻击者读取敏感配置文件、凭据等系统文件
    - PoC步骤：
        1. 向目标发送POST请求到/upload端点上传测试文件
        2. 从响应中提取文件下载路径
        3. 构造路径遍历payload读取系统文件（如/etc/passwd或windows/win.ini）
        4. 验证响应是否包含预期的文件内容
    - 修复建议：升级Gradio到版本3.33或更高（对于Gradio < 3.x）或版本4.11或更高（对于Gradio 4.x）
