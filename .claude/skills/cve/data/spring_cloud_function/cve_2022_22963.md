验证漏洞存在方法
    - 发送包含恶意SpEL表达式的HTTP POST请求到/functionRouter端点，如果服务器返回500状态码且触发了DNS/HTTP交互，则说明存在漏洞

漏洞利用方法
    - Spring Cloud Function存在SpEL表达式注入漏洞，攻击者可以通过构造特殊的routing-expression头来执行任意代码
    - POST /functionRouter HTTP/1.1
